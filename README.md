# Kaels-Map-Tool
Upload a map and add markers to it on various layers in a multi-user enviroment

My friends and I always feel like we need a more interactive map for the games we play.
Something that allows us to add an image of a map, and then together, at the same time even, edit that image.

Really this can be used as a second screen map for you and your friends in the same game, allowing you to even communicate and ping.

Layers
Add various layers for different categories of your markers.
Show and hide them as you please

Markers
Stylize and add markers to the map that when clicked, opens a Notes panel for the marker.

Host and Join
Share the map so multiple people (4 atm) can edit the map together in real time.
